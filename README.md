# bootcamp-session-5
Repository for bootcamp session 5 assignment

The website has 3 main pages which include:
-Home
-Clubs
-About

In the Clubs page, you can view 5 different clubs each displayed in a card with a short description and a 'View Detail' button.
The cards are shown in a grid formation by default, however user can also change it to a list view using the dropdown. The clubs can be sorted either ascending (A-Z) or descending (Z-A) by a dropdown.
Clicking on the 'View Detail' button opens the detail page, where you can see a longer description of the club along with upcoming club events.
There is a button which is used for joining the club. Once joined, the page will tell you that you are already part of that club.

Made with:
-React
-Tailwind CSS
